## How to Use

1. Browse the categories above to find libraries or packages you might find useful.
2. Each entry typically contains:
   - Name
   - GitHub repository link
   - Crates.io / npm / package registry link
   - Short description
3. Feel free to suggest new entries via pull requests or issues.

## Maintenance

This collection is maintained personally. Pull requests are welcome to add new libraries or update existing entries.
